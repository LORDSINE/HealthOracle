# Health Oracle

### Setup this project

```
git clone "https://github.com/LORDSINE/HealthOracle" HealthOracle

# make a branch before making changes on the code
git checkout -b branch_name

# To run the project

- For Windows:
    python -m venv .env
    .\.env\Scripts\activate

    python app.py

- For Linux/MacOS:
    python3 -m venv .env
    source ./.env/bin/activate

    python3 app.py
```
