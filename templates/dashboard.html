Early Risk Prediction for Heart Disease and Diabetes using Machine Learning

Health Oracle is an academic project that explores how machine learning models can assist in predicting the risk of heart disease and diabetes using health-related data.

About This Project

This application demonstrates the complete machine learning workflow from data exploration to model deployment. It analyzes shared risk factors between heart disease and diabetes, allowing for joint analysis and prediction.

The project compares multiple classification algorithms and explores how lifestyle and health indicators can be used for early risk assessment.
